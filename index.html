# Introdução A privacidade dos usuários do nosso aplicativo é muito importante para nós, e estamos comprometidos em protegê-los. Esta política explica sobre o uso de informações. O nosso aplicativo, serve apenas como uma ferramenta para instagramers, oferecendo facilidades e agilidades para os usuários. Ao usar o aplicativo, afirma estar de acordo com a política de privacidade. # Conteúdo público Este aplicativo não utiliza nenhum dado particular dos usuários, são exibidos apenas conteúdo público fornecido pelo Instagram, afins de exibição de imagem publicada, nome, username, foto de perfil, descrição da publicação. Com isto, possibilitamos que o usuário possa salvar a imagem adquirida através de uma URL da publicação. Para poder acessar as imagens da publicação, o perfil dos usuários precisam ser público, se foi utilizada URL de publicação de usuários com perfil privado, não poderemos acessar o conteúdo da publicação. Uso de ferramentas do dispositivo ## Armazenamento do dispositivo Pedimos autorização para uso de armazenamento apenas para salvar as imagens em seu dispositivo, não usamos para nada além disto. Com o pedido de permissão, o usuário poderá escolher se irá autorizar ou não, porém, caso não seja autorizado, o aplicativo não conseguiria salvar as imagens em seu dispositivo, pois, por conta da política de privacidade da Google, só pode se usada informação com o consentimento e autorização dos usuários. ## Uso do Clipboard O clipboard é usado apenas para colar a url copiar no Instagram, esse dado não fazemos nenhum armazenamento, esta ferramenta é fornecida pelo sistema operacional do dispositivo. Uso de anúncios Fazemos o use de anúncios no aplicativo para gerar renda, pois somos sem fins lucrativos. # Doações Para doações, utilizamos a plataforma PagSeguro, oferecemos 2 forma de doação, que seria, fazer uma doação só ou doação recorrente. nelas terá 3 opções de valor para escolher, são a de R$ 5,00, R$ 10,00 ou de R$ 15,00. Na doação única, tam a opção para doar qualquer valor.
